# Al Shifa Aayush Backend

This is the backend for the Al Shifa Aayush healthcare platform.

## Features
- Node.js + Express server
- CORS configured for frontend
- Health check endpoint at `/health`
- Global error handling
- Request logging with morgan

## Next Steps

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   npm run dev
   ```
